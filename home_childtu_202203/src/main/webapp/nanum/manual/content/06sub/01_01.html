관리페이지 보안

1. 관리자페이지 접근 관리

  • [관리자페이지 접근관리] 메뉴를 클릭해 주세요.
  • 관리자페이지 접근제한 설정을 해주세요. IP 접근제한과, 모바일기기 접속 설정이 가능합니다.
  • IP접근제한 사용 시 IP를 등록하시려면 원하시는 IP를 입력하시고 메모(업체명 혹은 IP에 관한 설명), 허용기간, 사용여부 선택 후 [등록] 버튼을 클릭해 주세요.
  • 등록하신 접근제한 IP 목록입니다. 해당 IP의 허용IP, 메모, 허용기간, 사용여부를 확인하실 수 있고 수정, 삭제 기능이 있습니다.
  • 여러개의 IP를 한꺼번에 삭제 혹은 사용여부변경이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 IP삭제 : IP를 선택하시고 [선택 IP삭제] 버튼을 클릭하시면 해당 IP가 일괄 삭제됩니다.
    선택한 IP 사용여부 변경 : IP를 선택하시고 사용여부 셀렉트 박스를 원하시는 것으로 선택 후 변경버튼을 누르시면 해당 IP의사용여부가 일괄 변경됩니다.
