삭제게시물 리스트

1. 삭제게시물 리스트 관리

  • [삭제게시물 리스트] 메뉴를 클릭합니다.
  • 찾으시는 게시물을 작성자, 게시판명, 제목으로 검색할 수 있습니다.
  • 삭제된 게시물 리스트입니다. 해당 게시물의정보와 복원, 삭제가 가능합니다.
  • 여러가지의 게시물을 한꺼번에 복원, 삭제 할 수 있습니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 게시글복원 : 게시물을 선택하시고 [선택 게시글복원]을 누르시면 해당 게시물이 일괄 복원됩니다.
    선택 게시글삭제 : 게시물을 선택하시고 [선택 게시글삭제]를 누르시면 해당 게시물이 일괄 삭제됩니다.
