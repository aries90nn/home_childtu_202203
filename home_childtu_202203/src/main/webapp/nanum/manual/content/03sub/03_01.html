배너관리

1. 배너리스트 페이지

  • [배너리스트] 메뉴를 클릭합니다.
  • 원하시는 배너를 제목 혹은 연결주소로 검색하실 수 있습니다.
  • 등록하신 배너의 목록입니다. 해당 배너의 제목, 사용기간, 사용여부를 확인하실 수 있고 수정, 삭제 기능이 있습니다.
  • 여러개의 배너를 한꺼번에 삭제 혹은 순서일괄수정, 사용여부변경이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 배너삭제 : 배너를 선택하시고 [선택 배너삭제] 버튼을 클릭하시면 해당 배너가 일괄 삭제됩니다.
    순서일괄수정 : 버튼을 클릭하시면 배너 순서를 일괄수정할 수 있습니다.
    선택한 배너 사용여부 변경 : 배너를 선택하시고 사용여부 셀렉트 박스를 원하시는 것으로 선택 후 변경버튼을 누르시면 해당 배너의 사용여부가 일괄 변경됩니다.

2. 신규 배너 생성

  • [신규 배너 생성] 메뉴를 클릭합니다.
  • 신규 배너의 정보를 입력해주세요.
    제목 : 배너의 제목을 입력해주세요. 배너 이미지의 alt 값으로 들어가는 부분이니 이미지에 있는 글들을 모두 적어주세요.
    사용기간 : 배너 개시의 시작일시와 종료일시를 선택해주세요. 계속 보이게 하시고 싶으시면 [무제한]을 체크해 주세요.
    배너 이미지 파일 : 배너 이미지를 선택하신 후 첨부해 주세요.
    연결주소 : 배너와 연결될 주소를 입력해 주세요.
    사용여부 : 배너의 사용여부를 선택해 주세요.
  • 신규 배너의 정보 입력 후 [등록] 버튼을 클릭하시면 신규 배너가 생성됩니다.
