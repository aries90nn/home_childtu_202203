디자인관리

1. 메인비쥬얼 리스트

  • [메인비쥬얼 리스트] 메뉴를 클릭해 주세요.
  • 원하시는 메인비쥬얼을 제목으로 검색하실 수 있습니다.
  • 등록하신 메인비쥬얼의 목록입니다. 해당 메인비쥬얼의 제목, 사용기간, 사용여부를 확인하실 수 있고 수정, 삭제 기능이 있습니다.
  • 여러개의 메인비쥬얼을 한꺼번에 삭제 혹은 순서일괄수정, 사용여부변경이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 메인비쥬얼삭제 : 메인비쥬얼을 선택하시고 [선택 메인비쥬얼삭제] 버튼을 클릭하시면 해당 메인비쥬얼이 일괄 삭제됩니다.
    순서일괄수정 : 버튼을 클릭하시면 메인비쥬얼 순서를 일괄수정할 수 있습니다.
    선택한 메인비쥬얼 사용여부 변경 : 메인비쥬얼을 선택하시고 사용여부 셀렉트 박스를 원하시는 것으로 선택 후 변경버튼을 누르시면 해당 메인비쥬얼의 사용여부가 일괄 변경됩니다.

2. 신규 메인비쥬얼 생성

  • [신규메인비쥬얼생성] 메뉴를 클릭해 주세요.
  • 신규 메인비쥬얼의 정보를 입력해주세요.
    제목 : 메인비쥬얼의 제목을 입력해주세요. 메인비쥬얼 이미지에 글자가 있으시면 이미지에 있는 글들을 모두 적어주세요. (이미지 alt값으로 필요)
    사용기간 : 메인비쥬얼 개시의 시작일시와 종료일시를 선택해주세요. 계속 보이게 하시고 싶으시면 [무제한]을 체크해 주세요.
    비쥬얼문구 : 비쥬얼에 들어가는 문구를 등록해 주세요.
    비쥬얼 이미지 파일 : 메인비쥬얼 이미지를 선택하신 후 첨부해 주세요.
    사용여부 : 메인비쥬얼의 사용여부를 선택해 주세요.
  • 신규 메인비쥬얼의 정보 입력 후 [등록] 버튼을 클릭하시면 신규 메인비쥬얼이 생성됩니다.
