3-2 팝업존관리

3-2-1 팝업존리스트 페이지

  • [팝업존 리스트] 메뉴를 클릭합니다.
  • 원하시는 팝업존을 제목 혹은 연결주소로 검색하실 수 있습니다.
  • 등록하신 팝업존의 목록입니다. 해당 팝업존의 제목, 사용기간, 사용여부를 확인하실 수 있고 수정, 삭제 기능이 있습니다.
  • 여러개의 팝업존을 한꺼번에 삭제 혹은 순서일괄수정, 사용여부변경이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 팝업존삭제 : 팝업존을 선택하시고 [선택 팝업존삭제] 버튼을 클릭하시면 해당 팝업존이 일괄 삭제됩니다.
    순서일괄수정 : 버튼을 클릭하시면 팝업존 순서를 일괄수정할 수 있습니다.
    선택한 팝업존 사용여부 변경 : 팝업존을 선택하시고 사용여부 셀렉트 박스를 원하시는 것으로 선택 후 변경버튼을 누르시면 해당 팝업존의 사용여부가 일괄 변경됩니다.

3-2-2 신규 팝업존 생성

  • [신규 팝업존 생성] 메뉴를 클릭합니다.
  • 신규 팝업존의 정보를 입력해주세요.
    제목 : 팝업존의 제목을 입력해주세요. 팝업존 이미지의 alt 값으로 들어가는 부분이니 이미지에 있는 글들을 모두 적어주세요.
    사용기간 : 팝업존 개시의 시작일시와 종료일시를 선택해주세요. 계속 보이게 하시고 싶으시면 [무제한]을 체크해 주세요.
    팝업존 이미지(메인) : 팝업존 이미지를 선택하신 후 첨부해 주세요.
    사용여부 : 팝업존의 사용여부를 선택해 주세요.
    연결창 : 팝업존과 연결될 주소 혹은 내용이 있으시면 입력해 주세요. 주소로 연결하시려면 페이지 체크, 이미지 혹은 별도의 내용으로 연결하시려면 팝업을 체크하신 후 내용을 적어주세요.
  • 신규 팝업존의 정보 입력 후 [등록] 버튼을 클릭하시면 신규 팝업존이 생성됩니다.
