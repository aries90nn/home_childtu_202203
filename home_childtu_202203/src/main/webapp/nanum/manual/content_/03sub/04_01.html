3-4 ebook관리

3-4-1 이북리스트 페이지

  • [이북리스트] 메뉴를 클릭합니다.
  • 원하시는 이북을 제목 혹은 내용으로 검색하실 수 있습니다.
  • 등록하신 이북의 목록입니다. 해당 이북의 제목, 사용여부를 확인하실 수 있고 수정, 삭제, 페이지관리 기능이 있습니다.
  • 여러개의 이북을 한꺼번에 삭제 혹은 순서일괄수정, 사용여부변경이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 이북삭제 : 이북을 선택하시고 [선택 이북삭제] 버튼을 클릭하시면 해당 이북이 일괄 삭제됩니다.
    순서일괄수정 : 버튼을 클릭하시면 이북 순서를 일괄수정할 수 있습니다.
    선택한 이북 사용여부 변경 : 이북을 선택하시고 사용여부 셀렉트 박스를 원하시는 것으로 선택 후 변경버튼을 누르시면 해당 이북의 사용여부가 일괄 변경됩니다.

3-4-2 신규 이북 생성

  • [신규 이북 생성] 메뉴를 클릭합니다.
  • 신규 이북의 정보를 입력해주세요.
    제목 : 이북의 제목을 입력해주세요.
    원본 이미지 사이즈 : 원본 이미지 사이즈를 Pixel단위로 입력해 주세요. 이북보기 화면에서 페이지의 확대 시 크기입니다.
    축소 이미지 사이즈 : 축소 이미지 사이즈를 Pixel단위로 입력해 주세요. 이북보기 화면에서 페이지의 축소 시 크기입니다.
    스킨설정 : 이북보기 화면의 스킨타입을 선택해 주세요.
    페이지 형태 : 이북의 페이지 형태를 선택해 주세요.
    로고 이미지 파일 : 이북보기 화면에서 상단에 로고를 첨부하실 수 있습니다.
    pdf파일 : 해당 이북의 다운로드가 가능하게 하시려면 pdf파일을 첨부해 주세요.
  • 신규 이북의 정보 입력 후 [등록] 버튼을 클릭하시면 신규 이북이 생성됩니다.

3-4-3 이북 페이지 설정

이북 리스트에서 해당 이북의 버튼 클릭하시면 이북 페이지 설정페이지로 이동됩니다.

  • [파일찾기] 버튼을 클릭하여 이미지를 등록해 주세요. 이미지 일괄등록 시 파일명의 순서대로 등록이 됩니다(예 : 가나다, ABC, 123 순)
  • 이미지 첨부 후 [등록] 버튼을 눌러주시면 이미지가 등록됩니다.
  • 등록하신 이북의 페이지 목록입니다. 해당 페이지의 페이지명, 이미지수정 및 해당 페이지 삭제가 가능합니다.
  • 여러개의 페이지를 한꺼번에 삭제 혹은 순서일괄수정이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 페이지삭제 : 페이지를 선택하시고 [선택 페이지삭제] 버튼을 클릭하시면 해당 페이지가 일괄 삭제됩니다.
    순서일괄수정 : 버튼을 클릭하시면 페이지 순서를 일괄수정할 수 있습니다.
이북으로 등록하고 싶으신데 pdf파일을 이미지로 변환하는 방법을 모르신가요?
pdf파일을 가지고 일괄적으로 이미지파일(jpg, gif 등)로 변환할 수 있습니다. 자세한 방법은 [나눔아이앤씨 꿀팁 바로가기]
