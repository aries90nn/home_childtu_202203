홈페이지 메뉴관리

1. 홈페이지 메뉴관리

홈페이지에 노출되는 메뉴의 등록·삭제와 사용여부(감추기 및 표시)를 간단하게 수정할 수 있습니다.

  • 메뉴명을 입력하고 사용여부를 선택하신 후 [등록] 버튼을 클릭하시면 메뉴가 등록됩니다.
  • 등록하신 메뉴의 목록으로 메뉴명과 해당메뉴에 대한 페이지확인, 메뉴명수정, 삭제 기능이 있고 사용여부를 알 수 있습니다.
  • 여러개의 메뉴를 한꺼번에 삭제 혹은 순서일괄수정, 사용여부 변경이 가능합니다.
    전체 선택/해제 : [전체 선택/해제] 버튼을 클릭하면 선택 체크박스가 모두 선택됩니다. 재클릭 시 선택 체크박스가 모두 해제됩니다.
    선택 메뉴삭제 : 메뉴를 선택하시고 [선택 메뉴삭제] 버튼을 클릭하시면 해당 메뉴가 일괄 삭제됩니다.
    순서일괄수정 : 버튼을 클릭하시면 메뉴 순서를 일괄수정할 수 있습니다.
    선택한 메뉴 사용여부 변경 : 메뉴를 선택하시고 사용여부 셀렉트 박스를 원하시는 것으로 선택 후 변경버튼을 누르시면 해당 메뉴의 사용여부가 일괄 변경됩니다.

2. 홈페이지 메뉴관리 - 하위메뉴 관리

왼쪽의 사이드바를 이용하여 2차, 3차메뉴를 추가하실 수 있습니다. 메뉴아이콘의 색상이 노랑색이면 사용여부 표시 상태, 회색이면 감추기 상태입니다.

  • 메뉴의 이름부분을 클릭하면 해당 메뉴의 하위메뉴가 아래에 나타납니다.
  • 버튼을 클릭하시면 ③번과 같은 메뉴설정 영역이 나타납니다.
  • 해당 메뉴의 하위메뉴관리, 컨텐츠관리, 컨텐츠보기, 이미지관리, css관리, 메뉴삭제가 가능합니다.
    하위메뉴관리 : 해당 메뉴의 하위메뉴관리로 현재가 1차메뉴라면 2차메뉴를 추가하실 수 있는창이 나옵니다.
    컨텐츠관리 : 해당 메뉴의 메뉴명, 입력유형, 작업내역, 하위메뉴 탭 사용여부, 메뉴표시여부, 실제URL, 내용을 등록할 수 있습니다.
    컨텐츠보기 : 해당 메뉴의 페이지 미리보기를 할 수 있습니다.
    이미지관리 : 해당 메뉴에 들어가는 이미지를 관리할 수 있습니다.
    css관리 : 해당 메뉴에 들어가는 css를 관리할 수 있습니다.
    메뉴삭제 : 해당메뉴를 삭제합니다.
  • 2차메뉴에서 메뉴명 옆에 아이콘이 붙었을 시 탭메뉴가 있다는 표시입니다.
