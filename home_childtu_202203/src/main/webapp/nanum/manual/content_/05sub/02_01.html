홈페이지 페이지관리

1. 홈페이지 페이지관리

홈페이지에 노출되는 해당메뉴의 페이지 설정입니다.

  • 해당 메뉴의 메뉴설정을 입력 및 선택해 주세요.
    메뉴명 : 메뉴명을 입력해 주세요.
    입력유형 : 직접입력(코딩형식으로 직접입력하는 방법), 게시판(등록된 게시판과 연동하는 방법), 링크(링크주소로 연결하는 방법), 유형없음(하위메뉴가 있을 시 선택)
    작업내역 : 작업내역이 남아서 전에 작업했던 내용을 확인할 수 있습니다.
    하위메뉴 탭 사용여부 : 하위메류들을 컨텐츠 안의 탭으로 구분할지 여부입니다.
    메뉴표시여부 : 홈페이지 상단/좌측 영역에 해당메뉴를 표시할지의 여부입니다.
    실제 URL : 실제 홈페이지에서 나오는 URL 주소 안내입니다.
  • 메뉴설정 완료 후 [미리보기] 버튼을 클릭하시면 설정된 화면을 확인할 수 있습니다.
  • 메뉴설정 완료 후 [등록] 버튼을 클릭하시면 설정하신 내용이 저장됩니다.
  • 입력유형 직접입력 체크 시 생성되는 입력란으로 내용을 넣으시면 됩니다.
  • 페이지 내용 입력 후 [등록] 버튼을 클릭하시면 내용이 등록됩니다.
